# bibliorevolucio
Biblioteca multimedia institut

Descripció:
Bibliorevolució es un projecte de programació pels alumnes del cicle superior de 
DAM (Desenvolupament d’aplicacions multiplataforma), aquest projecte consisteix en crear un programa, 
pel bibliotecari, que serveixi per controlar la biblioteca.
